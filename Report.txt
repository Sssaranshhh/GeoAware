GeoAware Project Report
======================

Overview
--------
GeoAware is a geospatial hazard monitoring and prediction system. It provides a REST API for predicting hazard levels based on seismic and weather data, and for retrieving recent seismic and weather trends. The backend is built with FastAPI (main API) and Flask (simple test frontend). Machine learning is used for hazard prediction.

How to Use
----------
1. Create a virtual environment and activate it.
2. Install dependencies from `requirements.txt`.
3. (Optional) Create a `.env` file with your API keys and paths.
4. Place your data file as `Final_data.csv` in the project root.
5. Run `python train_model.py` to train the model.
6. Start the API with `python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000`.
7. (Optional) Start the Flask frontend with `python app.py`.
8. Use the endpoints as described in `README.md`.
